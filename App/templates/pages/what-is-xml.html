What is XML?

Extensible Markup Language (XML) is a markup language that defines a set of rules for encoding documents that are human and machine-readable. 1998 specifications and many other related specifications: all open for free The XML design goals emphasize simplicity, generality, and user-friendliness on the Internet. It is a text data format with strong Unicode support for various human languages. Even though XML design focuses on documents, and the language is often used for arbitrary display Data structures as used in web services. Many schematic schemes can be used to define XML-based languages while programmers do Several APIs that have been developed to support the processing of XML data. see more in Wikipedia
